
# Logistica MVP Phase 2

Working MVP architecture:
- React frontend
- Express backend
- JWT-ready auth structure
- Product catalog
- Cart
- Quote requests
- Inventory API
- Order API
- Delivery tracking API
- PostgreSQL schema
- Docker Compose
- Images folder
- Rendering for all plattfrorms

Run:
1. npm install (frontend and backend)
2. Configure PostgreSQL
3. npm run dev
